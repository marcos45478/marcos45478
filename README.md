## 👨🏻‍💻 Marcos Vinicius

**`Desenvolvedor Full Stack`**

Sou estudante de Técnico em Informática para Web, focado no desenvolvimento de aplicações web práticas e funcionais. Tenho experiência com Python (Flask) no back-end e HTML, CSS e JSON no front-end, criando sistemas com cadastro de usuários, validação de dados (como CPF), autenticação de login e organização de informações.

Trabalho aplicando conceitos de Programação Orientada a Objetos (POO) e boas práticas de organização de código, buscando sempre desenvolver soluções simples, eficientes e bem estruturadas. Também utilizo Git e GitHub para versionamento de projetos e tenho noções de metodologias ágeis, como XP (Extreme Programming).

Estou em constante evolução, aprendendo novas tecnologias e aprimorando meus projetos para me tornar um desenvolvedor web cada vez mais completo.
